This is the documentation for the Data Visualization App, a powerful tool built with Flask that allows you to upload, clean, analyze, and visualize your data. This documentation will guide you through all aspects of the app, from initial setup to advanced usage.

Key Features:

  • Data Upload: Supports various file formats, including CSV, Excel, JSON, and more.

  • Data Quality Checks: Automatically identifies missing values, duplicate rows, and data type inconsistencies.

  • Data Cleaning: Provides tools to handle missing data and remove duplicates.

  • Interactive Visualizations: Generates a variety of plots based on your data and allows for customization.

  • AI-Powered Insights: Uses the Gemini language model to suggest visualizations and provide interpretations.

  • User-Friendly Interface: Designed for ease of use, even for users with limited data analysis experience.
