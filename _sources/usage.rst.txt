Introduction
============

This is the official documentation for **Data Visualization App**.

Installation
============

To install the application, run:

.. code-block:: bash

   git clone https://github.com/your-repo/dataviz-m2-iasd.git
   cd dataviz-m2-iasd
   pip install -r requirements.txt

Usage
=====

To run the application:

.. code-block:: bash

   python app.py
